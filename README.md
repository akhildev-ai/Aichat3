# AI Chatbot Frontend

Next.js frontend for the AI Chatbot application.

## Deploy to Vercel

1. Push this to a GitHub repo
2. Go to [vercel.com](https://vercel.com) → New Project → Import your repo
3. Vercel auto-detects Next.js — just add one environment variable:
   - `NEXT_PUBLIC_API_URL` = your Render backend URL (e.g., `https://aichat-backend-xxxx.onrender.com`)
4. Deploy!

## Local Development

```bash
npm install
npm run dev
```

Open http://localhost:3000
